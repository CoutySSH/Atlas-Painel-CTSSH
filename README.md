# Atlas Painel CTSSH

Painel web Atlas para VPS — gerencia SSH, checkusers, renovacoes e muito mais.

## Sistemas Suportados

| Sistema | Versao |
|---------|--------|
| Ubuntu  | 18.04, 20.04, 22.04, 24.04 |

> Testado apenas no Ubuntu. Nao testado em Debian, CentOS, Fedora ou outras distros.

## Tutorial de Instalacao

### 1. Baixar o instalador

```bash
wget https://raw.githubusercontent.com/CoutySSH/Atlas-Painel-CTSSH/main/install.sh
```

### 2. Dar permissao de execucao

```bash
chmod +x install.sh
```

### 3. Executar o menu

```bash
sudo ./install.sh
```

### 4. Menu principal

Apos executar, voce vera um menu numerado. As opcoes principais sao:

- **01** — Instalar dependencias (Apache, PHP, MariaDB, etc.)
- **02** — Configurar dominio e SSL (Certbot)
- **03** — Instalar Painel Atlas (baixa do GitHub, configura BD, permissoes)
- **04** — Configurar cron
- **05** — Resumo da instalacao

> Execute as opcoes em ordem (01 → 02 → 03) para uma instalacao completa.
